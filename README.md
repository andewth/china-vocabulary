# HSK5 China Focus Vocabulary

## GitHub Pages

1. Run `prepare-github.bat` (builds `docs/index.html`)
2. Push to GitHub
3. **Settings → Pages → Build from branch → `/docs` → Save**

URL: `https://YOUR_USERNAME.github.io/REPO_NAME/`

## Local use

- **With server:** double-click `start.bat` → http://127.0.0.1:8765/
- **Offline:** open `standalone.html` in browser (run `prepare-github.bat` or `python build_standalone.py` first to refresh)

Checklist is saved in browser localStorage on GitHub Pages, or in `checklist.json` when using the local server.
